# Vimba Python API Custom Library
---

`
2021.08.13
Ivan Gadjev
`

This library is built on the [Vimba Python API](https://github.com/alliedvision/VimbaPython).

The main purpose is to incorporate additional features specific to using the Manta GigE series camera for autocorrelator pulse-length measurements of the Magma25 laser at RadiaBeam.
